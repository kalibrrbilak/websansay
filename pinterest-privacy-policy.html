Политика конфиденциальности

Настоящая Политика конфиденциальности описывает, какие данные мы собираем, как они используются, и какие меры предпринимаются для их защиты при использовании нашего приложения, интегрированного с Pinterest через n8n. Мы соблюдаем требования конфиденциальности и защищаем личные данные наших пользователей.

Какие данные мы собираем

Мы можем собирать следующие данные для работы нашего приложения:

  • Имя пользователя Pinterest для авторизации и автоматизации публикаций.
  • Данные контента: текст, изображения, метки и ссылки, предоставляемые пользователем для публикации.
  • Технические данные: IP-адрес, cookie-файлы (если применимо) для обеспечения стабильной работы приложения.

Мы не собираем и не храним данные, которые не связаны с функциями нашего приложения.

Для чего используются данные

Собранные данные используются исключительно для выполнения следующих задач:

  • Автоматизация публикаций контента на платформе Pinterest от имени пользователя.
  • Обеспечение корректной работы приложения (например, обработка ошибок и улучшение производительности).
  • Предоставление пользователю доступа к функциям приложения.

Мы не используем данные для целей, не связанных с функциями приложения, и не передаем их третьим лицам без согласия пользователя.

Передача данных третьим лицам

Мы не передаем личные данные пользователя третьим лицам, за исключением следующих случаев:

  • Если это требуется для выполнения функций приложения (например, взаимодействие с Pinterest через API).
  • Если это необходимо для выполнения требований законодательства.

Во всех остальных случаях данные пользователя остаются конфиденциальными и не передаются третьим лицам.

Хранение данных

Собранные данные хранятся только в течение времени, необходимого для выполнения функций приложения.

  • Данные для автоматизации публикаций (текст, изображения, ссылки) обрабатываются в реальном времени и не сохраняются на серверах.
  • Технические данные, такие как лог-файлы, могут временно храниться для диагностики ошибок и улучшения работы приложения.

Мы применяем современные методы защиты данных, включая шифрование и защиту от несанкционированного доступа.

Права пользователей

Пользователи имеют следующие права в отношении своих данных:

  • Запрашивать информацию о том, какие данные собираются и как они используются.
  • Требовать удаления или изменения своих данных.
  • Отказаться от использования приложения и прекращения обработки данных в любое время.

Для реализации этих прав пользователи могут связаться с нами по электронной почте: info@kalininlive.ru.
